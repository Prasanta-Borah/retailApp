# retailApp
Run The application through command prompt from src folder
set class=JAVA_HOME/"bin";
set classpath=JRE_HOME/lib/rt.sr
javac com\retial\control\RetailMain.java
java com.retial.control.RetailMain
then enter the mobile number
after that it shows the item list
enter the item name as list above
after that enter the price
if want to continue then press y otherwise n
